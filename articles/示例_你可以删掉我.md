# 这是一篇示例文章

把你的新文章放在 `articles/` 目录下，命名成 `任意名字.md`，开头保留上面那段
`---` 包起来的 frontmatter（标题 / 分类 / 关键词 / 日期），下面正文用 Markdown 随便写。

写完运行：

```
python3 publish.py
```

就会自动并入知识库并打包出 `dist/index.html`，双击即可本地打开查看。

> 分类 `cat` 可选值见 `python3 add.py --list-cats`，或直接用网页里「✍ 写一篇新文章」按钮，
> 填好后点「导出 .md 文件」，把下载到的文件丢进 `articles/` 即可。
