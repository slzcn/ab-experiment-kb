📢 广告投放实验

投放管理

📄 2,279 字 · 🕐 2024-02-06

投放管理模块有两大功能，一是创建项目和广告；二是管理广告的投放数据，接下来我们仔细为您介绍这两部分能力

<span id="_1-创建项目和广告"></span>

1. 创建项目和广告

Tester是利用巨量引擎提供的MAPI进行项目和广告的创建，也就是说在Tester创建项目和计划的同时，会同步在巨量引擎后台同步创建。

<span id="_1-1-创建项目"></span>

创建项目

创建项目一共分为账户选择/营销目标设置/投放内容设置/投放版位设置/用户定向设置/排期与预算设置/搜索块头设置/监测链接设置等八个模块

<span id="5068e46c"></span>

账户选择

<span> </span>

参数配置说明
投放渠道默认选择“巨量引擎”。
账号选择选择已创建的广告账户。如果您是管理员角色，则您将获取所有授权账户的权限；如果您是普通成员角色，则您可以选择自己授权账户以及别人分配给到您的账户。添加广告账户的操作详情请参见添加与管理广告账户。
创建方式默认为新建项目，引用已有项目可以直接选择之前创建的项目。

<span id="56e8db23"></span>

营销目标与场景

<span> </span>

参数配置说明
推广目的支持应用推广、销售线索、电商店铺、小程序（当前支持微信小程序）。
营销场景当前只支持短视频/图片。
广告类型支持通投广告和搜索广告两种方式。

<span id="12d52487"></span>

投放内容与目标

  • 应用推广：

<span> </span>

参数配置说明
应用在下拉框中选择需要推广的应用。下拉框中展示“资产\-应用管理”中的全部启用的应用包。
下载方式配置后续广告实验过程中，广告推广目标人群如何下载应用，支持直接下载和落地页下载两种方式。
优化目标在下拉框中选择通过广告实验优化的目标。<br><br>下拉列表中展示的是要您在巨量后台“事件管理”中创建资产和事件，DataTester侧通过MAPI接口将巨量后台的资产拉取过来展示。
  • 销售线索/电商店铺/小程序

<span> </span>

参数配置说明
投放内容销售线索/电商店铺支持橙子落地页；小程序支持微信小程序。
优化目标不同的推广场景支持的优化目标不一致，可根据界面下拉框选择。

<span id="b37b38d0"></span>

投放版位

  • 通投智选：系统自动为您筛选投放广告位

<span> </span>

  • 首选媒体：您可以指定广告投放位

<span> </span>

<span id="666c36d6"></span>

用户定向

您可以在用户定向部分配置后续广告实验的目标推广人群。

<span> </span>

其中：

  • 自定义人群：可单击自定义人群，跟进界面提示，选择此前已添加至DataTester资产中的人群包。
  • 保存为元件包：这里DataTester增加了元件包的概念，已配置好的目标用户可保存为一个元件包，后续创建广告实验时支持直接从已有元件包获取一致的目标人群配置。

<span id="84063384"></span>

排期与预算

<span> </span>

参数配置说明
投放时间支持“从今天起长期投放”以及“设置开始和结束日期”两种方式
投放时段支持“不限”和“指定时段”两种方式
竞价策略支持“稳定成本”、“最优成本”和“最大转化”三种方式
项目预算支持“不限”和“设置预算”两种方式

<span id="bd45d1dc"></span>

搜索快投

<span> </span>

  • 通过MAPI直接拉取已选账户下的所有关键词
  • 出价系数：自定义输入

<span id="10378f62"></span>

第三方监测链接

<span> </span>

  • 已有链接：您可以在此下拉框中选择巨量引擎中创建的巨量引擎侧的监测链接。销售线索、电商店铺、小程序不支持。
  • 自定义：如果您希望使用DataTester侧的监控链接，可点击自定义页签，输入DataTester侧的监测链接。

<div data-tips="true" data-tips-type="warning" data-tips-is-title="true">注意</div>

  • <div data-tips="true" data-tips-type="warning">DataTester侧的监测链接需要在DataTester的广告资产中已添加为监测链接资产。</div>
  • <div data-tips="true" data-tips-type="warning">使用巨量引擎的监测链接时，监测的数据可通过巨量引擎的广告监测链接同步至巨量引擎侧，再通过DataTester侧的广告监测链接同步至DataTester侧；使用DataTester侧监测链接时，可在根据监测链接的配置，决定是否需要将监测数据回传给巨量引擎侧的监控链接。</div>

<span id="4776c47d"></span>

创建广告

创建广告一共分为推广身份设置/广告素材选择/广告设置/搜索流量设置/广告预算与出价/广告名称等6个步骤，广告和项目中的用户定向一样，可以作为元件创建多个，即代表可以同时创建多个广告，提高广告创建的效率。

<span> </span>

<span id="b26478a8"></span>

推广身份

<span> </span>

  • 推广身份：一共有“抖音号”和“账户信息”两种方式。
  • 抖音号选择需要您在巨量后台为广告账户绑定抖音号，我们直接拉取您账户下可以使用的抖音号。

<span id="9d80aee9"></span>

广告素材

  • 添加素材：

<span> </span>

点击添加素材会打开资产管理中的素材库，您可以直接选择素材库中的素材，上限10个素材。

  • 其他素材内容：

<span> </span>

还有文案标题/文本摘要/原生锚点设置/落地页/产品名称/产品主图/产品卖点/行动号召等元素。

<span id="93542ea2"></span>

广告设置

<span> </span>

  • 广告评论：是否开启用户评论。
  • 动态创意：是否开启智能动态创意。

<span id="7ce7244f"></span>

搜索流量

<span> </span>

  • 智能拓流：默认启用。
  • 关键词：点击编辑可以选择系统推荐的关键词。

<span id="270a429c"></span>

广告预算与出价

  • 竞价策略/优化目标/深度优化目标/计费方式等字段是直接从巨量引擎项目中拉取过来。
  • 预算/出价/深度转化出价：您可以根据自己的需求填写。

<span id="_2-投放数据管理"></span>

2. 投放数据管理

投放管理目前主要有项目和广告两个维度

<span> </span>

<span id="_2-1-筛选项"></span>

筛选项

项目维度下一共有“时间筛选/广告账户/推广目的/营销场景/计费方式/包含广告（广告名称）/项目状态/项目名称”八个筛选项；广告维度下一共有“时间筛选/广告账户/推广目的/营销场景/所属项目/项目状态/广告状态/广告名称”八个筛选项。其中，时间的选择默认为小时级的当日数据，支持切换为天级；日期选择，目前最多可选连续120天。

<span id="_2-2-批量操作"></span>

批量操作

选中项目或者广告，可以对其进行批量修改。

项目可以批量修改的内容有：启用/禁用/删除/时段/时间

广告可以批量修改的内容有：启用/禁用/删除/出价/预算

<span> </span>

<span id="_2-3-自定义指标"></span>

自定义指标

点击自定义列可以展示更多的指标选择，我们将指标分为属性设置/基础指标/事件指标/互动指标/行业指标五大类，右侧滚动列可以调整指标展示的顺序
