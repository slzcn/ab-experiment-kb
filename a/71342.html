📢 广告投放实验

新建广告实验

📄 2,441 字 · 🕐 2024-02-06

完成广告账户、广告资产等前期准备工作后，您可以在DataTester上创建广告实验，本文为您介绍新建广告实验的操作详情。

<span id="1b03f243"></span>

前提条件

  • 已完成前期准备工作：
  • DataTester侧：包括集团/DataTester项目、DataTester的旗舰版广告实验功能的产品认证等。
  • 巨量引擎侧：包括有巨量引擎的账号、创建巨量引擎的事件资产完成联调等。

详情请参见广告实验概述。

  • 已完成广告账户的授权添加，详情见添加与管理广告账户。
  • 已完成广告资产的添加，例如广告视频或图片的素材、广告落地页等，详情见添加与管理资产。

<span id="_1-输入基本信息"></span>

创建广告实验

<span id="0061346f"></span>

进入创建广告实验页面

登录DataTester控制台后，在左侧导航栏单击场景能力\>广告场景\>广告实验，进入广告实验页面后单击右上角的新建实验按钮，进入广告实验创建页面。

<span> </span>

<span id="f8f3a4f5"></span>

输入基本信息

输入实验名称、实验描述，实验负责人默认是创建者，实验时长默认30天。

<span> </span>

<span id="_2-填写项目信息"></span>

填写项目信息

<span id="386992bb"></span>

账户选择

<span> </span>

参数配置说明
投放渠道默认选择“巨量引擎”。
账号选择选择已创建的广告账户。如果您是管理员角色，则您将获取所有授权账户的权限；如果您是普通成员角色，则您可以选择自己授权账户以及别人分配给到您的账户。添加广告账户的操作详情请参见添加与管理广告账户。
创建方式默认为新建项目，引用已有项目可以直接选择之前创建的项目。

<span id="e0d07179"></span>

营销目标与场景

<span> </span>

参数配置说明
推广目的支持应用推广、销售线索、电商店铺、小程序（当前支持微信小程序）。
营销场景当前只支持短视频/图片。
广告类型支持通投广告和搜索广告两种方式。

<span id="07a3af8b"></span>

投放内容与目标

  • 应用推广：

<span> </span>

参数配置说明
应用在下拉框中选择需要推广的应用。下拉框中展示“资产\-应用管理”中的全部启用的应用包。
下载方式配置后续广告实验过程中，广告推广目标人群如何下载应用，支持直接下载和落地页下载两种方式。
优化目标在下拉框中选择通过广告实验优化的目标。<br><br>下拉列表中展示的是要您在巨量后台“事件管理”中创建资产和事件，DataTester侧通过MAPI接口将巨量后台的资产拉取过来展示。
  • 销售线索/电商店铺/小程序

<span> </span>

参数配置说明
投放内容销售线索/电商店铺支持橙子落地页；小程序支持微信小程序。
优化目标不同的推广场景支持的优化目标不一致，可根据界面下拉框选择。

<span id="69a81ee7"></span>

投放版位

  • 通投智选：系统自动为您筛选投放广告位

<span> </span>

  • 首选媒体：您可以指定广告投放位

<span> </span>

<span id="81bfa564"></span>

用户定向

您可以在用户定向部分配置后续广告实验的目标推广人群。

<span> </span>

其中：

  • 自定义人群：可单击自定义人群，跟进界面提示，选择此前已添加至DataTester资产中的人群包。
  • 保存为元件包：这里DataTester增加了元件包的概念，已配置好的目标用户可保存为一个元件包，后续创建广告实验时支持直接从已有元件包获取一致的目标人群配置。

<span id="03506e18"></span>

排期与预算

<span> </span>

参数配置说明
投放时间支持“从今天起长期投放”以及“设置开始和结束日期”两种方式
投放时段支持“不限”和“指定时段”两种方式
竞价策略支持“稳定成本”、“最优成本”和“最大转化”三种方式
项目预算支持“不限”和“设置预算”两种方式

<span id="2841666c"></span>

搜索快投

<span> </span>

  • 通过MAPI直接拉取已选账户下的所有关键词
  • 出价系数：自定义输入

<span id="f8ffe48c"></span>

第三方监测链接

<span> </span>

  • 已有链接：您可以在此下拉框中选择巨量引擎中创建的巨量引擎侧的监测链接。销售线索、电商店铺、小程序不支持。
  • 自定义：如果您希望使用DataTester侧的监控链接，可点击自定义页签，输入DataTester侧的监测链接。

<div data-tips="true" data-tips-type="warning" data-tips-is-title="true">注意</div>

  • <div data-tips="true" data-tips-type="warning">DataTester侧的监测链接需要在DataTester的广告资产中已添加为监测链接资产。</div>
  • <div data-tips="true" data-tips-type="warning">使用巨量引擎的监测链接时，监测的数据可通过巨量引擎的广告监测链接同步至巨量引擎侧，再通过DataTester侧的广告监测链接同步至DataTester侧；使用DataTester侧监测链接时，可在根据监测链接的配置，决定是否需要将监测数据回传给巨量引擎侧的监控链接。</div>

<span id="e8f88a36"></span>

配置实验内容

广告内容填写时，需要填写不同的广告版本，在不同实验版本中建议只替换广告的素材内容（建议每一个广告只添加一个素材，AB实验需要控制变量），可以添加多个实验组，目前对照组+实验组的总数上限是5组。具体广告实验各版本中的广告相关的详细配置参数与巨量引擎的广告创建配置一致。

<span id="1e766771"></span>

推广身份

<span> </span>

  • 推广身份：一共有“抖音号”和“账户信息”两种方式。
  • 抖音号选择需要您在巨量后台为广告账户绑定抖音号，我们直接拉取您账户下可以使用的抖音号。

<span id="7c7d727a"></span>

广告素材

  • 添加素材：

<span> </span>

点击添加素材会打开资产管理中的素材库，您可以直接选择素材库中的素材，上限10个素材。

  • 其他素材内容：

<span> </span>

还有文案标题/文本摘要/原生锚点设置/落地页/产品名称/产品主图/产品卖点/行动号召等元素。

<span id="8819b603"></span>

广告设置

<span> </span>

  • 广告评论：是否开启用户评论。
  • 动态创意：是否开启智能动态创意。

<span id="78a71dd3"></span>

搜索流量

<span> </span>

  • 智能拓流：默认启用。
  • 关键词：点击编辑可以选择系统推荐的关键词。

<span id="a4996096"></span>

广告预算与出价

  • 竞价策略/优化目标/深度优化目标/计费方式等字段是直接从巨量引擎项目中拉取过来。
  • 预算/出价/深度转化出价：您可以根据自己的需求填写。

<span id="_3-实验列表"></span>

查看实验状态与实验结果

  • 实验状态：实验计划创建之后进入到排期中，因为广告计划需要审核通过后才能投放出去，如果审核过程中计划没有通过，则状态切换为不满足实验条件；实验审核通过后则进入到实验进行中的状态。
  • 实验结果：实验对广告曝光量有限制，如果曝光数低于10000，会显示数据量不足。
